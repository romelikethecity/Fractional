Contractor Special Access Program Security Officer (CSSO)/Assistant Facility Security Officer (AFSO)

Dayton, OH, US

About This Role

Modern Technology Solutions, Inc. (MTSI) is seeking an experienced Assistant Facility Security Officer (AFSO) / Contractor Special Access Program Security Officer (CSSO) to support and help lead the Dayton Security Team. This role supports security operations for MTSI’s Dayton Industrial Security Program under the National Industrial Security Program (NISP) and provides subject matter expertise across Collateral, SAP, and SCI security disciplines.

The AFSO/CSSO assists the Facility Security Officer (FSO) and Site Security Manager (SSM) in ensuring compliance with applicable U.S. Government and corporate security requirements while safeguarding national defense information.

Key Responsibilities

Security Program Oversight

  • Assist in oversight and administration of classified security programs in compliance with:
  • 32 CFR Part 117 (NISPOM)
  • DoDM 5205.07 (SAP Security Manual)
  • DoDM 5105.21 (SCI Manual)
  • ICD/ICS 705 Technical Specifications
  • Director of National Intelligence Directives
  • MTSI security policies, processes, and procedures
  • Serve as a subject matter expert across collateral, SAP, and SCI security functions.
  • Maintain liaison with government agencies, customers, contractors, and internal leadership.

Personnel Security (PERSEC)

  • Support the FSO and SSM in managing personnel security clearance processes and investigations.
  • Conduct onboarding and termination security actions for cleared employees.
  • Facilitate badging and fingerprinting (including SWFT, as required).
  • Provide cross-functional onboarding support with HR and Recruiting.
  • Conduct candidate pre-screenings for security suitability.
  • Access, review, and submit clearance and access actions via DISS and NBIS.
  • Review and process documentation for SCI and SAP nomination packages.
  • Maintain personnel security records in SIMS, MTSI Virtual Personnel Records (VPR), and applicable government databases.
  • Prepare and administer security indoctrinations and debriefings.
  • Implement and manage visitor control procedures.
  • Manage SEAD 3 incident reporting, including submission to DCSA/AVS and appropriate cognitive security organizations.

Facility & Physical Security

  • Assist with daily secure facility operations, including:
  • Access control systems
  • Badging
  • Intrusion Detection Systems (IDS)
  • Support accreditation and maintenance of:
  • Collateral Open Storage Areas (OSA) per NISPOM and DCSA requirements
  • SAPFs and SCIFs per ICD/ICS 705
  • Assist with physical, personnel, information systems, network, and cryptographic security compliance.

Document Control & Asset Management

  • Support classified document control and accountability programs.
  • Manage and maintain:
  • GSA containers and locks
  • Secure storage equipment
  • COMSEC (as assigned)
  • Conduct annual inventories of accountable classified holdings.
  • Administer classified material destruction programs.
  • Serve as the focal point for receiving, processing, distributing, transferring, and controlling classified materials (hardcopy and electronic).
  • Prepare and submit required status reports to government agencies.

Security Operations & Compliance

  • Maintain security program records and documentation.
  • Support self-inspections, compliance assessments, and external audits.
  • Manage DD Form 254s, Security Classification Guides (SCGs), and classified contract requirements.
  • Analyze, investigate, resolve, and report security incidents and violations.
  • Provide guidance and leadership to employees and managers on security compliance actions.
  • Manage government-furnished equipment (GFE) and materials.
  • Respond to data calls and support security information systems.

Security Education, Training & Awareness (SETA)

  • Assist the FSO and Senior Site Security Manager in developing and delivering:
  • Initial and annual refresher training
  • Counterintelligence Awareness
  • Operations Security (OPSEC)
  • Program-specific and specialized security training
  • Ensure documentation and execution of all classified security training activities.

General Requirements

  • Strict adherence to government and corporate security directives.
  • Maintain confidentiality of:
  • Classified information
  • Company proprietary data
  • PII and CUI
  • Ensure compliance with contractual security requirements and DD Form 254 guidance.
  • Assist project managers and staff with interpretation and application of SCGs.
  • Implement and manage security incident response and reporting procedures.
  • Support enterprise-wide initiatives led by the Chief Security Officer (CSO), Chief Information Security Officer (CISO), and Director of Corporate Security (DCS).

Required Qualifications

  • Thorough knowledge of:
  • 32 CFR Part 117 (NISPOM)
  • SEAD 3
  • Industrial Security requirements
  • Proficiency with:
  • DISS
  • NBIS / eApp
  • NISS
  • Strong organizational, communication, and interpersonal skills.
  • Proven reliability, integrity, and trustworthiness.
  • Ability to develop and execute collateral security policies, plans, and procedures.

Desired Skills

  • Working knowledge of:
  • DoDM 5205.07 (SAP Manual)
  • DoDM 5105.21 (SCI Manual)
  • ICD/ICS 705 Technical Specifications
  • Experience supporting DoD SAP and/or SCI programs
  • SIMS database and SWFT fingerprinting experience
  • Export Control / ITAR experience
  • Knowledge of information systems security policies and procedures
  • Ability to develop and execute SAP and SCI security policies, plans, and procedures

Education & Certification

  • Bachelor’s degree with 3 years AFSO/FSO experience, or
  • 8 years total security experience in lieu of a degree
  • CDSE FSO Program Management for Possessing Facilities Curriculum
  • Must be obtained within 90 days of start date

Clearance Required: Active TS (SCI eligibility required)
